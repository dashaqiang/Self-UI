# Button 按钮组件

## 组件介绍

用于触发操作的按钮。

## 基本用法

```vue
<SelfButton type="primary">主要按钮</SelfButton>
```

## Props

| 属性名   | 说明         | 类型     | 默认值   |
| -------- | ------------ | -------- | -------- |
| type     | 按钮类型     | string   | default  |
| size     | 按钮尺寸     | string   | medium   |
| block    | 是否块级显示 | boolean  | false    |
| disabled | 是否禁用     | boolean  | false    |
| loading  | 是否加载中   | boolean  | false    |
